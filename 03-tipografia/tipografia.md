# 03 · Tipografía

## Familia tipográfica

Cáritas Argentina utiliza tipografías **sans-serif** en sus comunicaciones digitales, priorizando la legibilidad y la accesibilidad. El estilo es limpio, moderno y sin ornamentos, acorde al tono institucional y humano de la organización.

### Tipografía recomendada para web

**Open Sans** (principal) + **Raleway** o **Montserrat** (titulares)

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Open+Sans:wght@300;400;600;700&family=Raleway:wght@400;600;700;800&display=swap" rel="stylesheet">
```

> **Nota:** Verificar con el equipo de sistemas de Cáritas Argentina la tipografía oficial del sitio actual, ya que puede estar sujeta a actualización.

---

## Pesos tipográficos

| Peso | Valor | Uso |
|---|---|---|
| Light | 300 | Textos extensos, materiales de lectura |
| Regular | 400 | Cuerpo de texto, párrafos |
| SemiBold | 600 | Subtítulos, énfasis, etiquetas |
| Bold | 700 | Títulos, botones, headlines |
| ExtraBold | 800 | Titulares de alto impacto |

---

## Jerarquía tipográfica

### H1 — Titular principal
```
Font: Raleway ExtraBold (800) o Bold (700)
Size: 40–56px (desktop) / 28–36px (mobile)
Case: Sentence case o MAYÚSCULAS según contexto
Color: #FFFFFF (sobre rojo/azul) o #CF2E2E (sobre blanco)
Ejemplo: "70 años alentando la esperanza"
         "CÁRITAS ES SOLIDARIDAD QUE TRANSFORMA"
```

### H2 — Título de sección
```
Font: Raleway Bold (700)
Size: 28–36px
Color: #CF2E2E o #004684
Ejemplo: "Nuestros programas" / "¿Cómo ayudar?"
```

### H3 — Subtítulo / Nombre de programa
```
Font: Open Sans SemiBold (600) o Raleway Bold
Size: 18–22px
Color: #333333 o #004684
Ejemplo: "Seguridad alimentaria" / "Primera infancia"
```

### Body — Texto de cuerpo
```
Font: Open Sans Regular (400)
Size: 16px
Line-height: 1.7
Color: #333333
```

### Caption / Texto secundario
```
Font: Open Sans Regular (400)
Size: 13–14px
Color: #666666
```

### Botones / CTAs
```
Font: Open Sans Bold (700) o SemiBold (600)
Size: 15–16px
Case: Sentence case
Color: #FFFFFF sobre rojo o azul
Ejemplo: "Donar aquí" / "Colaborá con Cáritas"
```

---

## Uso en impresión

Para piezas gráficas impresas donde no se disponga de las fuentes digitales:
- Reemplazar **Raleway** por **Gill Sans** o **Futura**
- Reemplazar **Open Sans** por **Arial** o **Helvetica**

---

## Reglas generales

- No mezclar más de 2 familias tipográficas en una misma pieza
- Mantener consistencia en los pesos: no usar Light para CTAs ni ExtraBold para texto largo
- El texto en botones siempre en blanco sobre color institucional
- Evitar texto centrado en párrafos extensos — alinear a la izquierda para mejor lectura
